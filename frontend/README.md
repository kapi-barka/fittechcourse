# My Fitness Trainer - Frontend

Современное веб-приложение для фитнеса на Next.js 14 с App Router.

## Технологии

- **Next.js 14** - React фреймворк с App Router
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Zustand** - управление состоянием
- **Axios** - HTTP клиент
- **Lucide React** - иконки

## Установка и запуск

### 1. Установка зависимостей

```bash
cd frontend
npm install
```

### 2. Настройка переменных окружения

Создайте файл `.env.local`:

```
NEXT_PUBLIC_API_URL=http://localhost:8000/api
```

### 3. Запуск dev сервера

```bash
npm run dev
```

Приложение будет доступно по адресу: `http://localhost:3000`

## Основные страницы

### Публичные страницы
- `/login` - Вход в систему
- `/register` - Регистрация

### Защищенные страницы (требуется авторизация)
- `/dashboard` - Главная страница с обзором
- `/programs` - Программы тренировок
- `/diary` - Дневник питания и замеров тела
- `/articles` - Статьи о фитнесе

### Админ-панель (требуется роль admin)
- `/admin` - Управление пользователями, статьями, упражнениями

## Компоненты

### UI Компоненты
- `Button` - Кнопка с различными вариантами
- `Card` - Карточка для контента
- `Input` - Текстовое поле
- `Textarea` - Многострочное текстовое поле

### Функциональные компоненты
- `Navbar` - Адаптивная навигация

## Стилизация

Используется Tailwind CSS с кастомной цветовой схемой:

- Основной цвет: Purple (#667eea)
- Акцентный цвет: Blue (#764ba2)
- Градиенты для кнопок и акцентов
- Адаптивный дизайн (mobile-first)
- Темная и светлая темы

## API интеграция

API клиент автоматически:
- Добавляет JWT токен к запросам
- Обрабатывает ошибки 401 (перенаправление на логин)
- Типизирует все запросы и ответы

Пример использования:

```typescript
import { programsAPI } from '@/lib/api'

const programs = await programsAPI.list()
```

## Управление состоянием

Используется Zustand для управления состоянием аутентификации:

```typescript
import { useAuthStore } from '@/store/authStore'

const { user, login, logout } = useAuthStore()
```
